# Русификаторъ (Reverse)

## Info

* **Category:** Reverse
* **Type:** Browser Extension (.crx)
* **Difficulty:** Medium

---

## Task

Министерство Цифрового Благочиния разработало браузерное расширение, заменяющее иностранные слова на русские. Необходимо проанализировать расширение и найти скрытый флаг.

---

## Initial Analysis

Распаковываем `.crx` архив и изучаем структуру проекта.

Основные файлы:

* `background/service-worker.js`
* `lib/detector.js`
* `data/dictionary.json`

---

## Step 1: Поиск ложных флагов

В `background/service-worker.js` обнаружены строки:

```js
v: "ZmxhZ3tuMHRfdGgzX2ZsNGdfeTB1X2wwMGtfZjByfQ=="
p: "Synt{ge4afy4g3_gu1f_z4lo3}"
```

Декодирование:

* Base64 → `flag{n0t_th3_fl4g_y0u_l00k_f0r}`
* ROT13 → `Flag{tr4nsl4t3_th1s_m4yb3}`

Это ложные флаги.

## Step 3: Анализ detector.js

В `lib/detector.js` обращаем внимание на функцию `f()`:

```js
if (12 === t.length && 28746 === function(e)
```

Условие выполняется, если:

* длина строки = 12
* hash = 28746

При выполнении вызывается функция `u()`.

---

## Step 4: Получение массива h (ключа)

Если массив `h` отсутствует, он извлекается из изображения:

```
icons/icon-128.png
```

Берутся значения первых 16 пикселей.

### Скрипт для извлечения:

```python
from PIL import Image

img = Image.open("icons/icon-128.png")
pixels = img.load()

key = []

for x in range(16):
    r, g, b, a = pixels[x, 0]
    key.append(r)

print(key)
```

Результат:

```python
[246, 197, 133, 51, 2, 10, 76, 233, 254, 187, 186, 88, 113, 202, 105, 238]
```

---

## Step 5: Получение массива i

Добавляем логирование в `detector.js`:

```js
console.log(i);
```

Запускаем:

```bash
node detector.js
```

Получаем:

```js
[8, 8, 8, 7, 7, 7, 7, 7, 7, 7, 8, 8, 8, 7, ...]
```

---

## Step 6: Получение флага

Копируем функцию `u()` и вызываем её:

```js
console.log(u(i, h))
```
